Apache Isis™ is a platform to let you rapidly develop domain-driven apps in Java.

This example app demonstrates the Incode' Alias Module, configured for a very simple and purposefully minimal application.

The application demonstrates the ability to associate Aliases to any (arbitrary) domain object. Each alias is qualified by both an AliasType (eg to represent some external system such as General Ledger or Document Management) and also an application tenancy path (eg to support multi-tenancy, eg across countries). The consuming application is required to implement mandatory SPI repositories for these two qualifiers.

The module depends on the poly module (polymorphic association pattern). For more details, see the github repo.
