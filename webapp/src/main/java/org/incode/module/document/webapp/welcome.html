Apache Isis™ is a platform to let you rapidly develop domain-driven apps in Java.

This example app demonstrates the Incode Document Module, configured for a very simple and purposefully minimal application.

The application demonstrates the ability to preview or to create and attach Documents to any (arbitrary) domain object, assuming that there is a DocumentTemplate available that will consume the content of the input domain object to create the resultant Document.

The actual rendering of the Document is pluggable using the Renderer interface.

For more details, see the github repo.
